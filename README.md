## 🔍 Key Insights
- Revenue is driven by **high-value customers**  
- **Rock, Latin, Metal** are the top-performing genres  
- Top markets: **USA, Canada, Brazil**  
- **Iron Maiden** is the leading artist

## 🚀 How to Use

### 1. Setup Database
- Open your SQL environment (SQL Server / SQLite)  
- Run:
```
Music Store.sql
```

### 2. Explore SQL Analysis
- Write or run queries on the database to analyze:
  - Customers  
  - Sales  
  - Genres  
  - Revenue  

### 3. Open Dashboards
- Open:
```
Music Store.xlsx
```
- Navigate between sheets  
- Click **Data → Refresh All** (if needed)

### 4. View Full Report
- Open:
```
Full Report.pdf
```
